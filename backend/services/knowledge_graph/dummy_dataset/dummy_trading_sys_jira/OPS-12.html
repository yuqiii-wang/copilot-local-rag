OPS-12

Collateral Management System

TypeStory
StatusDone

Assignee: Charlie

Description

Users need to post collateral to satisfy margin requirements.


Features

- Define `CollateralType` (CASH, TBILL, GOLD).

- `Collateral` object with haircut properties.

- `CollateralManager` to track posted assets.

- Method to `checkMarginCall` comparing Total Collateral Value vs Maintenance Margin.


Acceptance

- Can add cash and bonds.

- Value calculates correctly (Amount * (1-Haircut)).
