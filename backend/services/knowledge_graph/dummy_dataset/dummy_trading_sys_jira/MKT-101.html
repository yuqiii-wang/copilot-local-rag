MKT-101

Implement Market Data and Pricing Interface

TypeTask
StatusDone

Description

We need a way to get prices for securities to value the portfolio and check limits.


Tasks

- Create `MarketData` DTO (Bid, Ask, Last, Timestamp).

- Create `Pricer` interface (getPrice, getValue).

- Implement `SimplePricer` back-end (in-memory map of latest prices).


Note

This is a prerequisite for Risk and Margin modules.
